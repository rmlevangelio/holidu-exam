# Holidu Exam

### Prerequisites

*Node version:* v8.10.0

*Npm version:* 5.6.0

In order to run this project:
```bash
# install the project's dependencies
yarn install
# Runs the app in development mode.
yarn start
# Generate production bundle
yarn run build
# run the unit tests with Jest
yarn test
# run eslint
yarn run lint
```
